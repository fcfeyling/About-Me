    Homelab Specs

    Below are the specs of my home server. Currently I only have one server in my homelab, but am looking to expand so I can experiment with high availability clustering.

    CPU

    Dual Xeon e5 2665.

    Motherboard

    SuperMicro X9DRD-EF dual-socket LGA2011 motherboard with Intel C602J chipset

    Memory

    32 GB 10600 DDR3 RDIMM (4 x 8GB)

    Boot Drive

    Samsung 860 EVO 500gb SATA SSD

    Storage

    Four 2TB Western Digital enterprise SATA drives configured in RAID-Z

    Network

    Dell Broadcom Quad Port 1Gb NIC
